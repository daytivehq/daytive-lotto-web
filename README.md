# 오늘의 로또 번호

매일 새로운 로또 번호를 추천받는 웹 서비스

**[lotto.daytive.com](https://lotto.daytive.com)**

## 기능

- 오늘의 추천 번호 생성
- 번호 다시 뽑기
- 즐겨찾기 저장 (브라우저 저장)
- 반응형 디자인 (모바일 지원)

## 기술 스택

- **Astro** + **React** + **TypeScript**
- **Tailwind CSS**
- **Cloudflare Pages**

## 로컬 실행

```bash
npm install
npm run dev
```

http://localhost:4321 접속

## 빌드

```bash
npm run build
```
